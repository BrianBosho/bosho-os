# Weekly Action Board

This page is the operational control layer for the BrainDump.

Use it to decide:

1. what the active workstreams are,
2. what the next action is for each one,
3. what fits into this week.

## How to use this page

- Keep only active workstreams here.
- Every row needs one next action and one concrete output.
- If a task is too large for a single work block, split it before scheduling it.
- Review this page daily and reset it weekly.
