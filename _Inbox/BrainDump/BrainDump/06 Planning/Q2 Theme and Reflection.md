# Q2 Theme and Reflection

Source: [[../00 Inbox/Workbook q2 wk1 - Raw Archive|Workbook q2 wk1 - Raw Archive]]

## Quarter theme through June 2026

- Concentration
- Competence
- De-procrastination / execution
- Expansion

## Interpretation from the workbook

### Concentration

- Define 3 key things to complete daily
- A thing usually takes 1.5 to 3 hours
- Each thing should have a concrete output

### Execution

- Complete at least 3 items that currently have resistance each day
- Items can be as small as a cold email or a phone call, but they should be completed

### Competence

- Allocate serious time to papers, tutorials, and learning resources
- Focus on a few major things rather than scattered browsing

### Expansion

- Deliberately work on new things beyond day-to-day obligations
- Identify multiple opportunities every week
- Commit to at least one application each week
- Precommit product features for weekly progress

## Reflections captured in the workbook

- A weekly cadence is needed
- Workflows are needed for identifying items, breaking them down, scheduling them, and following through
- The same execution system should support applications, product work, and admin tasks

## Quarter operating principles

| Theme | Working rule | Visible behavior |
|---|---|---|
| Concentration | choose only a few important outputs each day | 3 meaningful outputs, not scattered activity |
| Competence | spend deliberate time learning deeply | scheduled reading, tutorials, and technical notes |
| Execution | finish resisted items instead of circling them | calls, emails, submissions, and decisions completed |
| Expansion | keep opening new paths while shipping current work | applications, opportunities, and product progress each week |

## Reflection checklist

- [ ] Pick the 3 outputs that define a good day.
- [ ] List the resisted items that must still get done.
- [ ] Reserve real learning time instead of ad hoc browsing.
- [ ] Commit to at least one growth action each week.
- [ ] Review whether the week reflected concentration or fragmentation.

## Weekly reflection prompts

| Prompt | Purpose |
|---|---|
| What were the 3 outputs that mattered most this week? | check concentration |
| Which resisted items did I finish? | check execution |
| What did I study deeply? | check competence |
| What new opportunities or features did I advance? | check expansion |
| Where did the week fragment? | identify structural failure points |
