# BrainDump README

# BrainDump

Central home for extracted notes from the Google Doc brain dump.

Main organizing page:
- [[BrainDump/BrainDump Dashboard|BrainDump Dashboard]]

## Structure
- `00 Inbox` — raw captures and unsorted notes
- `01 Opportunities` — jobs, PhDs, fellowships, applications workflow
- `02 Research` — VLM Grounding, FedProp, multimodal audio, literature, experiments
- `03 Projects` — College App, Swyng, product ideas
- `04 Operations` — OpenClaw, systems, workflows, tooling
- `05 Admin` — transactions, accounts, logistics
- `06 Planning` — weekly planning, priorities, scheduling, execution system

This folder is meant to keep all brain-dump-derived material in one place, while still grouping it into usable subfolders.
