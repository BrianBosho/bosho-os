# Today

This is the daily launch page. It is the first page to open each morning.

Use it to choose today's work, process the inbox, and open the daily record. Keep this page stable; dated entries live in [[Planning/Daily Log]].

## Start Here

> [!summary] Daily command center
> Start with orientation, choose a small number of actions, then work from artifacts rather than pressure.

| Area | Link | Use |
|---|---|---|
| Daily record | [[Planning/Daily Log]] | Add today's dated section and shutdown review. |
| Selected actions | [[Planning/To-Do]] | Pull today's 3-5 actions from the short-term pool. |
| Weekly shape | [[Planning/This Week]] | Check the week's priorities and constraints. |
| General dashboard | [[General/Dashboard]] | Review active work without opening every project. |
| Home dashboard | [[Home/Dashboard]] | Check life systems, recovery, and non-work commitments. |

## Operating Rules

> [!important] Rules to live by today
> - Use the early hours for the highest-value thinking before the mind and world become noisy.
> - Prefer one real artifact over a vague work session.
> - If a plan feels self-deceptive after two honest minutes, shrink it.
> - If something can be fixed by structure, do not over-analyze it.
> - Reliability before output. Recovery before optimization. Closure before expansion.

## Inbox Processing

> [!todo] Inbox rule
> The inbox is for capture, not storage. Process items into projects, planning, home systems, archive, or trash.

Use [[_Inbox/Unprocessed]] for raw material and unprocessed thoughts.

When processing an item, choose one outcome:

| Outcome | Destination |
|---|---|
| Project state or context | Relevant `General/Projects/<project>/<project>.md` or `execution.md` |
| Near-term action | [[Planning/To-Do]] |
| Today's action or note | [[Planning/Daily Log]] |
| Life / health / personal system | [[Home/Dashboard]] or the relevant Home note |
| Reference only | Appropriate Knowledge folder or archive |
| Not useful | Delete or leave for later cleanup |

## New Day Protocol

1. Open [[Planning/Daily Log]] and add a new dated section at the top.
2. Pull 3-5 actions from [[Planning/To-Do]].
3. Decide the first artifact before opening extra notes.
4. Process only enough inbox material to remove friction.
5. At shutdown, move outcomes to the right place: completed items, project execution notes, carry-forward, waiting, or parked.
