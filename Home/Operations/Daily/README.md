# Daily

Reflective journaling only — one living file instead of one note per day:

- **[[Journal]]** — rolling, dated reflection. Newest on top. Each entry is a short *distilled* paragraph (what mattered and what it means), not a raw dump. Durable ideas graduate to permanent notes and are linked.

Tasks and execution do **not** live here. They live in the [[Dashboard|Planning Dashboard]] (Today / This Week / Weekly Plan) and the project execution boards. Keep this folder for reflection so the two don't drift.

Principle: no empty stubs, no raw entries. If a reflection grows into something durable, move it to a permanent note (e.g. `Knowledge/Mind/`) and leave a link.
