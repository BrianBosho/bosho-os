# Decision Rules

> Hard rules that override scoring. Read before making any apply/skip decision.

## Hard Stop Rules (Always Override Score)

1. **No visa path = No application.** If immigration_score = 0, do not proceed regardless of other scores.
2. **Deadline passed = Archive immediately.** No exceptions.
3. **Unfunded PhD = Archive.** Brian cannot afford an unfunded PhD.
4. **Explicitly "citizens only" = Archive.** Do not pursue.
5. **Compensation below floor = Skip** (unless exceptional strategic value that is clearly documented).

## Apply Rules

6. Score ≥ 8.5 → Must apply. Begin drafting immediately.
7. Score 7.0–8.4 → Apply. Medium or lower effort only unless strong strategic case for high effort.
8. Score 5.5–6.9 → Human decision required. Do not draft until Brian decides.
9. Score < 5.5 → Skip unless: (a) near-zero effort (quick apply), AND (b) documented strategic reason.

## Effort Rules

10. Do not invest High or Very High effort on score < 7.0.
11. Do not invest Medium effort on score < 6.0.
12. Low-effort applies (score 5.5+) are acceptable as a scatter strategy.

## Urgency Rules

13. Deadline < 7 days → Flag as CRITICAL immediately regardless of score.
14. Deadline 7-14 days → Flag as URGENT. Prioritize over other tasks.
15. Deadline > 60 days → Can be queued; do not block urgent items.

## Duplication Rules

16. Never create a note for an opportunity already in Application_History.
17. Never apply to the same position twice (unless explicitly reopened).

## Reference Check Rules

18. Do not submit any application requiring reference letters without confirming referee availability first.
19. Request letters at least 3-4 weeks before deadline; prefer 6+ weeks.

## Quality Rules

20. Never send a motivation letter with generic phrases — each application must have at least one paragraph specific to that institution/lab/company.
21. Always run quality checklist in Agent_Application_Drafter.md before finalizing any document.
