# Agent Guide - Draft Application

## Goal
Draft tailored application materials using the canonical knowledge base.

## Read first
- relevant profile notes
- relevant project evidence notes
- relevant CV blocks and letter blocks
- fit/appraisal summary for the opportunity

## Rules
- do not invent experience or results
- reuse only evidence grounded in the knowledge base
- tailor emphasis to the opportunity type
- for PhD roles, foreground research coherence and academic strength
- for ML jobs, foreground implementation strength and applied systems delivery

## Outputs
- tailored CV draft
- tailored cover/motivation letter draft
- optional outreach email
- optional portfolio project shortlist
