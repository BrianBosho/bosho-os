# Agent Guide - Appraise Opportunity

## Goal
Determine whether an opportunity is a good fit for Brian and how strongly it should be prioritized.

## Read first
- [[01_Profile/Profile Snapshot]]
- [[02_Positioning/Research Interests]]
- [[03_Skills-and-Fit/Fit Profile]]
- [[04_Mobility-and-Preferences/Mobility Preferences]]
- [[07_Appraisal-Rules/Quality Rubric]]

## Questions to answer
1. Does this align with Brian's real interests?
2. Does he qualify directly, partially, or only as a stretch?
3. Does his profile offer a distinctive advantage here?
4. Are visa/funding/logistics feasible?
5. Is this strategically important enough to pursue?

## Output
- scores for each rubric dimension
- summary of why it fits / does not fit
- risks or gaps
- recommendation: must apply / strong apply / conditional / skip
