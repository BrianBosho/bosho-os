# AI Tools Inventory

## Current / Intended Tools

- ChatGPT Planner: planning, reflection, weekly review, protocols, drafts.
- OpenClaw: agentic execution, coding/research workflows.
- OpenAI Codex / Codex CLI: codebase inspection, scripts, refactors, debugging, tests, experiment automation.
- Minimax M3: supporting model for OpenClaw workflows.
- Google Docs/Drive: human-readable notes and shared documents.
- GitHub / local repos: code, markdown Wiki, experiment systems.

## Tool Principle

Use agents for friction, not avoidance.

Agents should reduce mechanical burden and increase clarity. They should not replace Brian’s judgment.

## What Agents Can Do

- Inspect codebases.
- Generate scripts.
- Aggregate results.
- Check table/config consistency.
- Draft documentation.
- Summarize papers.
- Create meeting agendas.
- Convert braindumps into tasks.
- Draft emails.
- Create weekly plans.

## What Brian Must Decide

- What matters.
- What is true enough to publish.
- What projects to accept/refuse.
- What work deserves prime cognition.
- What kind of life is being built.
