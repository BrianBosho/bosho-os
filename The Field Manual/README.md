# PhDMaxxing OS Vault

This is an Obsidian-ready starter vault for planning and running the PhD as a life-and-research operating system.

## How to use

1. Unzip this folder.
2. Open Obsidian.
3. Choose **Open folder as vault**.
4. Select the unzipped `PhDMaxxing_OS_Vault` folder.
5. Start from `PhDMaxxing OS.md` or `00_Home/Home.md`.

## Core rule

Every note should serve at least one of five purposes:

1. Clarify direction.
2. Produce research artifacts.
3. Improve feedback loops.
4. Track opportunities.
5. Keep you alive and agentic.

If a note does none of these, do not create it yet.

## Day-one workflow

- Open `00_Home/Home.md` every day.
- Capture unsorted thoughts in `Inbox.md`.
- Maintain only one active project unless explicitly decided otherwise.
- Run `00_Home/Weekly Review Template.md` every week.
- Use `02_Projects/Anti-Diffusion Protocol.md` whenever an idea feels interesting but vague.
