# Workday Protocol

## Purpose

A workday is for bounded, high-quality execution.

## Default Shape

| Time | Activity |
|---|---|
| 7:30 | Wake |
| 7:30–8:15 | Hygiene, light movement, breakfast/coffee if needed |
| 8:15–8:30 | Work setup ritual |
| 8:30–11:30 | Deep Work Block 1 |
| 11:30–12:30 | Food + walk/rest |
| 12:30–3:30 | Deep Work Block 2 |
| 3:30–5:00 | Workout/cardio + shower |
| 5:00–6:00 | Meal/rest |
| 6:00–9:00 | Deep Work Block 3 or support block |
| 9:00–9:30 | Shutdown ritual |
| 9:30–10:30 | Meditation, reading, wind-down |
| 10:30–11:00 | Sleep |

## Success Levels

- Minimum viable workday: 1 deep block + clean shutdown.
- Full workday: 2 deep blocks + 1 support block.
- Heroic workday: 3 deep blocks.

## Evening Rule

Evening work should usually be light/admin/support work unless there is a deadline.

Good evening work:

- Email.
- Planning.
- Documentation.
- Meeting prep.
- Result organization.
- Reading light material.
- Updating project pages.

Avoid:

- Starting new experiments.
- Deep theoretical work.
- Scope expansion.
- Emotional debugging.
