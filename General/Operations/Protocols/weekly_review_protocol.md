# Weekly Review Protocol

## When

Sunday afternoon/evening, or Friday shutdown for work-specific review.

## Inputs

- Weekly plan.
- Project pages.
- Inbox notes.
- Calendar.
- Health/recovery observations.
- Open loops.

## Questions

### Mission

- What was this week’s mission?
- Was it achieved?
- What actually happened?

### Projects

- Which projects moved?
- Which projects are stuck?
- Which projects should be paused, delegated, or removed?
- What is the next action for each active project?

### Recovery

- How was sleep?
- How was exercise/cardio?
- How was diet/snacking?
- How was mood/energy?
- What signals should be monitored?

### Work Hygiene

- Were work blocks bounded?
- Did I define outputs clearly?
- Did I shut down cleanly?
- Did work leak into rest days?

### Next Week

- What is the weekly mission?
- What are the top 3 outcomes?
- What are the 12 possible work blocks?
- What must be protected?

## Output

A completed weekly plan for the next week.
