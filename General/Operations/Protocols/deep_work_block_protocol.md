# Deep Work Block Protocol

A work block is treated like an exam session: prepared, bounded, serious, and cleanly closed.

## Standard Block

Duration: 3 × 50 minutes, with short breaks.

## Before the Block

Answer:

1. What is the exact output?
2. What materials must be open?
3. What is forbidden?
4. What counts as done?
5. Where does this task go next if unfinished?

## Setup

- Clear desk.
- Open only necessary tabs/files.
- Put phone away.
- Start timer.
- Write block objective at top of scratchpad.

## During the Block

- Stay with the defined output.
- Do not expand scope.
- Capture side thoughts in scratchpad; do not pursue them.
- If stuck, write the obstacle explicitly.

## End of Block

Record:

- Completed:
- Remaining:
- Decision made:
- Next action:
- Next scheduled slot:

## Block Quality Check

A good block produces one of:

- A corrected section.
- A table.
- A script.
- A notebook.
- A message.
- A decision.
- A document.
- A clear diagnosis.

Not acceptable:

- “Worked on paper.”
- “Read some things.”
- “Thought about it.”
