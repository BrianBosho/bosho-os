# Vault Usage Rules

## Daily

- Capture raw thoughts in `00_inbox`.
- Use work block template for serious work.
- Shutdown before bed.

## Twice weekly

- Process inbox.
- Update active projects dashboard.
- Move tasks into weekly plan.

## Weekly

- Run weekly review.
- Create next weekly plan.
- Choose 12 possible work blocks.
- Protect 3 restoration days.

## Monthly

- Review whether the summer mission is still correct.
- Archive dead projects.
- Update AI workflows.
