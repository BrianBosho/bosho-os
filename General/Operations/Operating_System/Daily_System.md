# Daily System

## Core Rule

Use the early hours for the highest-value thinking before the mind and world become noisy.

## Minimum Viable Daily Shape

- Sleep early enough to protect the morning.
- Decide the first task the night before.
- Start with one concrete deep-work block.
- Keep the first block bounded and artifact-oriented.
- Avoid using the morning for logistics or indecision.

## What Matters Most

- Predictable rhythm.
- Fewer decisions.
- Fast entry into real work.
- Real closure at the end of the day.

## Anti-Drift Rule

If you wake up and still need to choose what matters, too much of the day is already lost.

## Weekly Operating Shape

- One main research direction per period, not many.
- One defined deep-work block per day.
- One collaboration move per week.
- One review point for open loops and next actions.
- One protected recovery half-day or full day that is not secretly optimization.

## First Principles

- Reliability before output.
- Recovery before optimization.
- Defaults before willpower.
- Closure before expansion.
- One real block of depth is worth more than a day of agitated pseudo-work.
