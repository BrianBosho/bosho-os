# Agent Instructions

## Purpose

This repository is a long-term personal codex and knowledge system.

## Principles

- Preserve structure and clarity.
- Prefer durable summaries over noisy logs.
- Keep uncertainty visible.
- Do not overwrite historical context casually.
- Use Markdown only.
- Maintain Obsidian-compatible links.

## Writing Style

- Calm
- Structured
- Evidence-aware
- Concise but meaningful

## Editing Rules

- Prefer additive edits.
- Use atomic notes where possible.
- Create backlinks between related concepts.
- Avoid redundant notes.
