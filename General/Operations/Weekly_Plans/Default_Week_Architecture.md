# Default Week Architecture

## Function

A stable weekly skeleton whose whole job is to remove the daily question *"What should I do now?"*. It converts the entire mass of commitments into recurring containers. You don't need to feel excited about every concern — you only need to arrive at the correct container.

> **The schedule decides the category; the first five minutes decide the exact output.**

This is the operational answer to the state described in [[Functional_Formulation]] and [[Five_Kleshas]]. See also [[Mind_OS_Map]].

## The default workday

**8:00–11:00 — Fedprop.** Same project, same time, every workday. No deciding whether you feel ready. Each morning has one defined output: results audit + experiment decisions / rewrite a specific set of tables / run or analyze experiments / revise the appendix / integrate and send the draft. Four or five mornings = **12–15 hours** — enough to move Fedprop from a frightening abstraction to a reviewable draft, even if it doesn't finish every possible improvement.

**1:00–4:00 — Second deep-work block.** Rotate the other substantive work here (VLM, Colab, Fedprop overflow).

**4:30–6:00/6:30 — Exercise block.** Work out, go out, hang out.

**7:30–9:00 — Logistics block.** Only one logistics theme per day. Don't enter with "handle logistics" — enter with something bounded, e.g. *"Find the decision deadline and draft the questions for Jaime"* or *"Establish the emergency visa-request procedure and identify the documents needed."*

## A workable week

**Monday** — 8–11 Fedprop audit + exact task list · 1–4 VLM project plan & delegation · 4:30 Andrew ID + set Portugal decision deadline

**Tuesday** — 8–11 Fedprop results rewrite · 1–4 VLM first technical contribution · 4:30 prepare questions for Jaime / current students

**Wednesday** — Recovery day. At most one genuinely urgent admin action. Do not use it to compensate for Mon/Tue.

**Thursday** — 8–11 Fedprop results/experiments · 1–4 VLM team-unblocking · 4:30 visa expedition process + school email

**Friday** — 8–11 Fedprop appendix, integration, draft package · 1–4 Colab or remaining Fedprop · 4:30 Portugal decision synthesis + next actions

This yields: **~12h protected Fedprop · ~9h VLM/Colab · ~6h contained logistics · one real recovery day.**

## The rules

- At 8:00 it is Fedprop because that is what the architecture says — not because it won the morning's urgency contest.
- At 4:30 the paper loses the right to address you; logistics owns that floor.
- **Five-minute shutdown** after the logistics block: (1) record what was completed, (2) write the next action, (3) confirm tomorrow's first output, (4) declare the workday closed.
