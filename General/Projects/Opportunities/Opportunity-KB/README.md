# Opportunity KB

This folder is an Obsidian-ready starter knowledge base for opportunity appraisal and application support.

## Architecture
- Obsidian KB: canonical truth about Brian, reusable assets, fit rules, and agent guidance
- Notion DB: high-volume opportunity inventory and pipeline state
- Agent repo: automation logic, scraping, appraisals, drafting, and syncing
