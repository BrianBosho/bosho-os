# FedProp

## One-line summary
Communication-efficient federated GNN framework for reconstructing missing-neighbor information via local feature propagation.

## Problem
Federated graph learning suffers from incomplete neighborhood information because clients do not share raw graph structure and features freely.

## What Brian did
- formulated and implemented a feature propagation-based solution
- developed training and evaluation pipelines
- connected the method to theoretical reasoning around propagation and reconstruction

## Methods / tools used
- PyTorch
- GNNs
- federated learning
- graph benchmarking

## Results / outcomes
- achieved 99.6% of centralized model performance on Cora without extra inter-client communication
