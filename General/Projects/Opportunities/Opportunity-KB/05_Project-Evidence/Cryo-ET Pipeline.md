# Cryo-ET Pipeline

## One-line summary
Deep-learning pipeline for tomogram denoising and automated particle picking in Cryo-Electron Tomography workflows.

## Problem
Cryo-ET data are extremely noisy and difficult to annotate, making automated analysis challenging.

## What Brian did
- developed an end-to-end deep-learning workflow
- integrated denoising and particle-picking components
- validated the pipeline on public EMPIAR datasets

## Methods / tools used
- 3D U-Net
- DeepETPicker
- scientific imaging workflows
- reproducible experimentation

## Results / outcomes
- reduced manual analysis burden and demonstrated applicability on real biological data
