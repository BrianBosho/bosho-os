# Quality Rubric

## Core scoring dimensions (1-10)
- Interest Fit
- Qualification Fit
- Unique Fit
- Immigration Feasibility
- Strategic Value
- Prestige / Upside

## Optional modifiers
- Application Effort
- Deadline Urgency
- Topic Excitement

## Suggested weighted score
Overall Priority =
0.25 * Interest Fit +
0.20 * Qualification Fit +
0.15 * Unique Fit +
0.15 * Immigration Feasibility +
0.15 * Strategic Value +
0.10 * Prestige / Upside

## Decision buckets
- 8.5-10.0: must apply
- 7.0-8.4: strong apply
- 5.5-6.9: conditional apply
- below 5.5: usually skip
