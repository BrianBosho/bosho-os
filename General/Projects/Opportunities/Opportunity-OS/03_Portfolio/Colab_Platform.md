# Colab: Multi-Provider LLM Chat Platform

## One-Line Summary
Production-grade AI chat platform with a FastAPI backend and Next.js frontend, abstracting across OpenAI, Anthropic, Gemini, and OpenRouter through a unified API layer with secure proxy patterns.

## Two-Line Summary (for CVs)
Built a multi-provider LLM chat system with provider abstraction across OpenAI, Anthropic, Gemini, and OpenRouter. Implemented secure proxy patterns, prompt/config management, and conversational service workflows; tested with pytest and deployed on Railway/Vercel.

## Problem
Developers and teams using LLMs face vendor lock-in and inconsistent APIs across providers. A unified abstraction layer enables provider-agnostic development, easier switching, and centralized key management.

## Brian's Role
- Full system design and implementation
- FastAPI backend with provider routing layer
- Multi-provider integration (OpenAI, Anthropic, Gemini, OpenRouter)
- Secure proxy patterns and API key management
- Conversational service design
- Frontend: Next.js chat interface
- Testing: pytest suite
